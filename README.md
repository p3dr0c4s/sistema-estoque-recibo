# Sistema de Estoque e Gerador de Recibo

Projeto simples de sistema de estoque com geração de recibo, desenvolvido com HTML, CSS e JavaScript. O sistema permite adicionar produtos, visualizar lista de itens cadastrados e gerar um recibo com total da compra e código de barras simulado.

## Funcionalidades

- Cadastro de produtos (nome, preço e quantidade)
- Listagem de produtos em estoque
- Remoção de itens do estoque
- Geração de recibo de compra
- Cálculo automático do valor total
- Geração de código de barras aleatório (simulado)

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)

## Como executar o projeto

1. Faça o download ou clone este repositório
2. Abra o arquivo `index.html` em qualquer navegador moderno
3. Utilize a interface para adicionar produtos e gerar recibos

Não é necessário instalar dependências.
